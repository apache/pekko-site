    3. Data Types & Abstractions

    HTTP and related specifications define a great number of concepts and functionality that is not specific to either HTTP’s client- or server-side since they are meaningful on both end of an HTTP connection. The documentation for their counterparts in Apache Pekko HTTP lives in this section rather than in the ones for the Client-Side API, Core Server API or High-level Server-Side API, which are specific to one side only.
